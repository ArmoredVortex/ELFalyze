# ELFalyze 🔍

A web-based ELF binary analysis tool powered by React and Node.js.

## Features

- ELF Header parsing
- Section & Symbol table display
- Disassembly view

## Tech Stack

- React (frontend)
- Node.js / Express (backend)
- Styled with SCSS

## Setup

```bash
# Frontend
cd client
npm install
npm start

# Backend
cd server
npm install
npm run dev
```
